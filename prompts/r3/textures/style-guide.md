# Stage III — Hotel Imperial
# Texture Style Guide

## CRITICAL OUTPUT REQUIREMENTS

Every texture must satisfy all of the following. Violating any one makes the output unusable:

1. **64 × 64 pixels exactly. Square. 1:1 aspect ratio.** Width = 64. Height = 64.
   Not 128. Not 256. Not rectangular. The raycaster hard-codes 64-pixel UV coordinates.
2. **Pixel art.** Not painted. Not photorealistic. Not watercolour. Not impressionist.
   Every pixel is a flat, aliased, single-colour square. No anti-aliasing. No smooth
   gradients. No brush strokes. No blur. Hard edges everywhere.
3. **PNG, 8-bit RGB, no alpha channel.**
4. **No post-processing filters.** No vignette, no bloom, no lens flare, no noise filter.

**Style reference:** Early 1990s first-person shooter game wall textures (Wolfenstein 3D /
DOOM shareware era). Hand-pixeled retro game tile assets. Each pixel individually placed,
not generated by a painting algorithm. Looks like a texture ripped from a 1993 shareware
FPS, not a painting.

---

## What these textures are for

A first-person raycasting horror game (Wolfenstein / DOOM renderer style).
The engine casts one ray per screen column. When a ray hits a wall it reads a single
vertical column of pixels from the wall texture. Floor and ceiling pixels are computed
per screen pixel using planar projection. All textures must be pixel art to match the
renderer's visual style and to sample correctly at integer UV coordinates.

**Wall textures** are sampled as a single vertical column (texX 0–63). The column is
stretched vertically to fill the on-screen wall strip height.
**Horizontal tiling must be seamless** — left and right edges must match exactly.
Vertical seamlessness optional.

**Floor and ceiling textures** are tiled across the plane using planar perspective mapping.
**Both axes must tile seamlessly.**

## Setting

**The Hotel Imperial, Longmont, Colorado — opened 1902, closed 1971, sealed since.**

A five-storey Edwardian hotel on the corner of Main and 3rd. Once the grandest accommodation
between Denver and Cheyenne. Sealed abruptly in 1971 — linen still on the beds, cutlery
still on the tables, keys still at the front desk. The building has not been maintained
since. Forty years of slow rot: wallpaper peeling from damp, mahogany darkened to near-black,
the gilt ballroom ceiling crumbling in pale gilded flakes.

## Pixel art specification

- **Every pixel is a deliberate, aliased, flat-colour square.** No sub-pixel rendering.
- Pixel clusters of 2–4 px wide represent one surface feature (a seam, a pattern repeat,
  a highlight edge, a crack).
- Shading is **banded** — discrete value steps with hard edges between them. No gradients.
- Colour palette per texture: **12–20 distinct colours maximum.**
- No anti-aliasing on any edge. No smoothing of any kind.
- The visual result must look like a hand-pixeled retro game tile, not a painting.

## Colour temperature

- Floors 0–1 (Lobby, Ballroom): very dark, with warm amber and dark gold undertones.
  The gilt surfaces glow faintly — slightly warm against cold stone.
- Floors 2 (Kitchen): cold, blue-white ceramic. Clinical and hostile.
- Floors 3–5 (Guest floors): cool blue-grey. The wallpaper was once warm but has
  faded to near-grey.
- Floors 6 (Honeymoon Suites): deep crimson warmth, heavy fabric.
- Floors 7–8 (Service areas): cold, functional. Dark wood and stone.
- Floor 9 (Penthouse): oppressively warm — the most gold, the most dark.

**Maximum brightness:** No surface brighter than 55–60% value across all tiles.
This is a horror game — everything is dark.
