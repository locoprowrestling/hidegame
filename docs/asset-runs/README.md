# Asset Run Notes

This folder records generated-asset work that is useful for later production
runs. Use one short note per batch and start from
[`docs/templates/asset-run.md`](../templates/asset-run.md).

Run notes are operational history, not a replacement for prompt files:

- update prompts when the desired output changes;
- update the production guide when the workflow changes;
- use run notes for commands, outcomes, verification, and lessons.
